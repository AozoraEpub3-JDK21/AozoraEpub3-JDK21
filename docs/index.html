AozoraEpub3 — ダウンロードとインストール

青空文庫形式テキストをEPUBに変換するツール。ダウンロード・セットアップ・起動の手順ガイド。

最新リリースを開く

必要な環境

  • Java 21 以降（Temurin等のJRE/JDKを推奨）
  • 対応OS: Windows / macOS / Linux

Windowsの手順（おすすめ）

  • Releasesから ZIP をダウンロードして任意フォルダに展開
  • フォルダ内の AozoraEpub3起動.bat または AozoraEpub3.bat をダブルクリック
  • うまく起動しない場合は Java 21 の導入 と bat経由の起動 を確認

macOS / Linuxの手順

  • Releasesから TAR.GZ をダウンロードして展開
  • ターミナルでフォルダに移動し、./AozoraEpub3.sh を実行
  • 権限エラー時は chmod +x AozoraEpub3.sh を実行してから起動

直接コマンドで起動（カスタム向け）

java -jar AozoraEpub3.jar -of -d out input.txt

GUIは 引数なし で java -jar AozoraEpub3.jar を実行すると起動します。

よくある質問（FAQ）

  • Windowsで.jarが開けない: バッチ（.bat）から起動してください。
  • Javaが見つからない: Java 21 の JRE/JDK をインストールしてください。
