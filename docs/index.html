AozoraEpub3 — ダウンロードとインストール

青空文庫形式テキストをEPUBに変換するツール。ダウンロード・セットアップ・起動の手順ガイド。

最新リリースを開く

必要な環境

  • Java 21 以降（JRE / JDK）
  • 対応OS: Windows / macOS / Linux

推奨: Eclipse Temurin から Java 21 LTS の JRE をダウンロード

Java 21 のインストール（Eclipse Temurin）

Windows

  1. Adoptium Releases を開く
  2. JDK 21 → Windows x64 → .MSI を選択してダウンロード
  3. .MSI ファイルをダブルクリックして実行し、ウィザードに従ってインストール
  4. ターミナル（コマンドプロンプト）で確認: java -version

macOS

  1. Adoptium Releases を開く
  2. JDK 21 → macOS → .PKG (Intel) または .PKG (Apple Silicon M1/M2) を選択してダウンロード
  3. .PKG ファイルをダブルクリックしてインストール
  4. ターミナルで確認: java -version

Linux（Ubuntu/Debian）

  1. Adoptium Releases を開く
  2. JDK 21 → Linux x64 → .TAR.GZ をダウンロード
  3. 任意のフォルダに展開: tar -xzf OpenJDK21U-jdk_x64_linux_hotspot_21_x.tar.gz
  4. PATH に追加するか、コマンド全パスで実行: ./jdk-21.x.x+yy/bin/java -version

Windows でのダウンロード・インストール（おすすめ）

  1. リリースページ から最新の ZIP をダウンロード
  2. ZIP を任意のフォルダ（例: `C:\Program Files` や デスクトップ）に展開
  3. フォルダを開き、AozoraEpub3起動.bat をダブルクリック
  4. GUI が起動したら準備完了

注: jar ファイルのダブルクリックが無反応の場合は、.bat ファイルをご利用ください

macOS / Linux でのダウンロード・インストール

  1. リリースページ から TAR.GZ をダウンロード
  2. 任意のフォルダに展開: tar -xzf AozoraEpub3-*.tar.gz
  3. ターミナルでフォルダに移動し、./AozoraEpub3.sh を実行
  4. 権限エラー時は先に実行: chmod +x AozoraEpub3.sh

CLI（コマンドライン）での実行

カスタマイズの詳細設定が必要な場合、コマンドラインから直接実行できます。

java -jar AozoraEpub3.jar -of -d out input.txt

引数の詳細は README をご参照ください。

GUI は 引数なし で起動：java -jar AozoraEpub3.jar

トラブルシューティング

  • Java がインストールされていない: Temurin から Java 21 JRE をダウンロード・インストール後に再度お試しください。
  • Windows の jar ファイルがダブルクリックで開かない: .bat ファイルを使用してください。または、ターミナルから java -jar AozoraEpub3.jar で起動できます。
  • Linux/macOS で permission denied エラー: chmod +x AozoraEpub3.sh を実行してから再度起動してください。
  • その他の問題: GitHub Issues でお知らせください。
